# Hexo博客重新起航

经过长时间的沉寂，这个博客终于重新开始建设了！

## 为什么选择Hexo

Hexo 是一个快速、简洁且高效的博客框架，使用 Markdown（或其他渲染引擎）解析文章，在几秒内即可生成静态网页。

### Hexo 的优势

1. **快速生成** - Hexo 基于 Node.js，性能极佳
2. **Markdown支持** - 使用 Markdown 编写内容，专注于创作
3. **丰富的主题** - 有众多精美主题可选
4. **易于部署** - 一键部署到 GitHub Pages

## 新的开始

这次重建博客，我选择了以下技术栈：

- **Hexo 7.x** - 最新的稳定版本
- **NexT 8.x** - 经典主题的最新版
- **响应式设计** - 适配各种设备
